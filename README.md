ZakApps Assessment (HTML, CSS, JavaScript)

This project is a simple e-commerce flow built using plain HTML, CSS, and vanilla JavaScript as part of an assignment.

Pages Included

Home Page – Click any product to go to product listing

Product Listing Page – Filter, sort, and product category filters

Product Detail Page – Add to Cart (updates cart count), Buy Now → Order Placed page

Cart Page – Shows added items and total amount

Features

Functional filter and sort

Add to Cart with quantity updates

Simple order placed flow

Clean folder structure with dummy product data

How to Run

Open index.html in your browser.
